SOC Threat Intelligence Dashboard
🚀 Project Overview

SOC Threat Intelligence Dashboard is an advanced cybersecurity monitoring dashboard developed using Python Flask, HTML, CSS, JavaScript, and Chart.js.

This project simulates a real-world Security Operations Center (SOC) environment where security analysts can monitor, analyze, and investigate suspicious activities, malicious IPs, and cyber threats in real time.

The dashboard provides:

Real-time incident monitoring
Threat severity analysis
Risk score visualization
MITRE ATT&CK mapping
Interactive threat intelligence
Security analytics graphs
Incident export functionality
IP-based threat investigation
🔥 Features
✅ Real-Time Incident Monitoring

Continuously fetches incidents from backend APIs and displays them live.

✅ Risk Score Analysis Graph

Visualizes attack risk scores using dynamic bar charts.

✅ Severity Distribution Chart

Displays High / Medium / Low threat distribution using doughnut charts.

✅ IP Threat Intelligence

Analyze suspicious IP addresses with:

Total alerts
Threat level
MITRE ATT&CK techniques
Risk percentage
✅ MITRE ATT&CK Integration

Maps alerts to:

Techniques
Tactics
ATT&CK framework categories
✅ Threat Timeline

Displays attack progression timeline for every incident.

✅ Live Search Filter

Search incidents using:

IP Address
Alert Type
MITRE Technique
✅ Export Incident Data

Download incidents as JSON reports.

✅ Professional SOC UI

Modern cybersecurity-themed dashboard with:

Dark mode UI
Responsive layout
Hover effects
Animated cards
Live updates
